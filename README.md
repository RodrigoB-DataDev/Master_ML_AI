## 🤖 Actividad 4 — Proyecto Final Integrador

El desarrollo más completo de este repositorio es la **Actividad 4**, el proyecto final del módulo *Python para IA*.

Se trata de una **aplicación web con IA** que integra tres herramientas de procesamiento de texto:

- 💬 **Análisis de sentimiento** — detecta el tono emocional de cualquier texto
- 📄 **Resumidor** — condensa textos largos en diferentes formatos
- 🌍 **Traductor inteligente** — traduce entre 12 idiomas con control de tono

👉 **Para ver la documentación completa, capturas y el link a la app desplegada, consulta el [README de la Actividad 4](./Actividad%204/README.md)**
